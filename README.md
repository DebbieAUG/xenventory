# Xenventory

A lightweight inventory management system built for internal teams.

## Features

* 🔐 Admin & User login
* 📦 Add inventory
* 📋 View inventory dashboard
* 📝 Request inventory items
* ✅ Admin approval workflow
* 📊 Live inventory summary
* 📜 Activity logging

## Tech Stack

* HTML
* Tailwind CSS
* Vanilla JavaScript
* Google Apps Script
* Google Sheets (Database)
* GitHub Pages (Hosting)

## Project Structure

```text
xenventory/
├── index.html
├── dashboard.html
├── css/
│   └── styles.css
├── js/
│   ├── auth.js
│   ├── dashboard.js
│   └── api.js
└── assets/
```

## Database

Google Sheets contains the following sheets:

* Inventory
* Requests
* Users
* Logs

## Deployment

* Frontend hosted on GitHub Pages
* Backend powered by Google Apps Script Web App
* Data stored in Google Sheets
